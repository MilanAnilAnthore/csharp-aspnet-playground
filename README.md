# csharp-aspnet-playground

A personal playground repository for practicing C# and ASP.NET (including common framework features and patterns).

This repo is used for small experiments, demos, and learning exercises—things like:
- C# language practice and syntax experiments
- ASP.NET app basics (routing, controllers, middleware, configuration)
- Trying out libraries/packages and different project setups
- Quick prototypes and proof-of-concepts

Nothing here is intended to be production-ready—it's mainly for learning and iteration.
